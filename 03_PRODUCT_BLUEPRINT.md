# 产品蓝图

## 产品定位

Gold Summit Capital 网站不是普通官网，也不是简单的基金后台。

它应被设计为：

> 服务高净值家族的投资与传承内容中枢。

客户登录后应该感知到：

1. Gold Summit Capital 专业、稳定、透明
2. 客户不仅获得基金管理服务，也获得持续的认知服务
3. 下一代可以通过平台逐步理解投资、风险、责任和家族传承

## 公共区域

### 目标

- 建立信任
- 呈现理念
- 保持低调和克制
- 引导客户进入专属区

### 页面建议

1. 首页
2. 关于 Gold Summit Capital
3. 团队
4. 理念
5. 客户登录

### 风格原则

- 不做夸张营销
- 不使用过度煽情的财富传承语言
- 强调专业、长期主义、信任、责任和连接
- 视觉上保持安静、稳重、有质感

## 客户专属登录区

### 1. Dashboard 首页

客户登录后的默认页面。

内容建议：

- 最新月报
- 最新基金表现摘要
- 最新内部研究
- 最新活动回顾
- 下一场活动预告
- 本月推荐阅读

设计目标：

- 一眼看到重点
- 不让客户迷路
- 像私人简报入口，而不是复杂后台

### 2. 基金信息

内容建议：

- 基金表现摘要
- 月度净值走势
- 累计收益
- 月度收益
- 最大回撤
- 策略说明
- 重要通知

注意：

基金数据、持仓和策略披露必须遵守合规边界。

### 3. 月报与研究

内容建议：

- 历史月报
- 内部研究报告
- 宏观观察
- 行业研究
- 持仓逻辑复盘
- AI 与科技趋势
- 家族财富与传承专题

运营原则：

- 宁可少而精，不要多而杂
- 每月稳定更新
- 每篇内容都要有明确观点和客户价值

### 4. 活动记录

活动记录采用 Event Capsule 活动胶囊格式。

每场活动沉淀为一篇结构化内容，而不是照片墙。

内容建议：

- 活动主题
- 时间与城市
- 活动背景
- 议程摘要
- 3-5 条核心洞察
- 嘉宾观点摘要
- 匿名化 Q&A 主题
- 推荐阅读
- 给家庭内部讨论的 3 个问题
- 现场非人物照片

### 5. 传承与教育

内容建议：

- 下一代投资基础
- 风险、杠杆、流动性和周期
- 家族沟通问题卡
- 家族治理入门
- AI 时代的职业、创业和投资机会
- 公益、影响力投资与社会责任
- 大陆、香港、台湾跨地区资源理解

形式建议：

- 短文
- PDF
- 活动后小结
- 推荐阅读清单
- 轻量课程笔记

## MVP 不包含

第一版暂不做：

- 客户之间互相聊天
- 社区论坛
- 复杂 App
- 在线报名系统
- 个性化投资分析
- 大量视频课程
- 客户之间公开互相可见的社交关系
