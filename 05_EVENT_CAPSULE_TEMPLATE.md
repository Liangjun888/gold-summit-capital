# Event Capsule 活动胶囊模板

## 使用目的

活动现场不适合拍摄客户正脸或披露客户身份，但公司仍然需要记录做过的事情，并把活动沉淀为客户可见的内容资产。

Event Capsule 的目标不是做照片墙，而是记录活动的思想密度、关系温度和专业价值。

## 可用图片素材

建议拍摄：

- 会场空间局部
- 桌面、资料、茶点、书籍
- 城市、建筑、场地细节
- 讲者背影或远景，前提是获得授权
- 屏幕内容，前提是去除敏感信息
- 白板或手写内容，前提是脱敏

避免拍摄：

- 客户正脸
- 客户姓名牌
- 合照
- 可识别客户身份的侧脸或背影组合
- 未授权嘉宾照片
- 具体投资敏感信息

## 发布前检查

- 是否移除客户姓名和身份信息？
- 是否移除客户可识别照片？
- 是否移除敏感投资信息？
- 嘉宾观点是否获得授权？
- 是否需要创始人或合规人员确认？
